# Welcome to CoreCartDocs

**CoreCart** is a project inspired by OpenCart, built from scratch using clean code and featuring fixes for known OpenCart issues!

This wiki is designed for developers and creators of online stores who want to understand the engine's architecture or contribute to the open-source community.
